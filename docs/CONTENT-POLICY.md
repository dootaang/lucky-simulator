# 콘텐츠 권리·배포 정책 (초안 — 오너 결정 대기 항목 포함)

> 상태: 2026-07-17 지휘자 초안. 원칙 절은 기존 ADR·로드맵에서 이미 확정된 내용의 재확인이고,
> 「오너 결정 대기」 절의 체크박스는 오너가 정하기 전까지 보수적 기본값으로 운영한다.

## 확정 원칙 (기존 결정의 재확인)

1. **원본은 사용자 로컬에만 있다.** 카드·에셋 팩은 사용자가 직접 가져오며, 럭키는 실행 환경만 제공한다
   (ROADMAP §리스크). 럭키가 카드 원본이나 에셋을 재배포하지 않는다.
2. **변환 결과는 기본적으로 그 기기 안에서만 쓴다.** 인증 변환판(SimPack)은 로컬 저장·로컬 플레이가 기본이다.
3. **`.simpack`은 설계상 원본 카드 바이트(sources)를 포함한다.** 따라서 **변환판 공유 = 원본 재배포**다.
   공유 기능을 열려면 그 전에 **sources 제외 내보내기**(규칙·화면·모듈 구성만 담은 SimPack 변형)가 먼저
   구현되어야 한다. 이 순서는 뒤집지 않는다.
4. **제작자 크레딧.** 인증 카드 문서에는 카드 제작자와 출처를 명시하고, "공식판" 표기는 제작자 허가를
   받은 뒤에만 쓴다(Risumari 전례: 공개 글 크레딧 의무 확인).
5. **상용 IP 표기.** 팬카드가 원작 게임(예: 소녀전선)의 이름을 쓰는 것과 별개로, 럭키의 대외 문구는
   "OO 카드 지원/호환"이지 "OO 게임 제공"이 아니다. 원작 IP 보유자와의 제휴를 암시하는 표현을 쓰지 않는다.

## 오너 결정 대기

- [ ] **로컬 캐시 복사 허용 범위**: 원본 에셋을 브라우저 저장소(OPFS/IndexedDB)에 복사·보존하는 현재
      방식 유지 여부 (기본값: 허용 — 사용자 기기 안이므로 재배포가 아님)
- [ ] **변환판 공유 기본값**: sources 제외 내보내기 구현 전까지 공유 기능 비노출 (기본값: 비노출)
- [ ] **성인 콘텐츠 변형 인덱싱**: 에셋 팩의 선택적 성인 이미지 그룹을 기본 인덱싱할지, 옵트인 토글로
      분리할지 (기본값: 인덱싱은 하되 화면 노출은 카드 선언을 따름 — 별도 필터링은 미구현)
- [ ] **제작자 허가 취득 시**: "인증 공식판" 배지와 갤러리 등재 조건 (Phase 3 커뮤니티 베타에서 재론)

## 인증 변환판의 정직성 계약 (구현됨, 2026-07-17)

- 인증 변환은 원본 수치를 **회수**하는 것이 기본이고, 회수 불가능한 규칙만 Lucky 규칙으로 **합성**한다.
- 회수/합성 구분은 컴파일 결과의 `warnings`(회수 요약)와 `issues[source=template]`(합성 목록)에 남기며,
  카드 열기 시 진단 콘솔에 자동 기록된다. 조용한 확정은 없다.
