# 브랜드 규범 — 럭키★시뮬레이터

> 확정: 2026-07-12 (오너 결정 + 펜타 수다 종합). 리네이밍 실행 계획은 이 문서 §5.

## 1. 이름

| 대상 | 이름 |
|---|---|
| 제품 | **럭키★시뮬레이터** (Lucky★Simulator) |
| 엔진 | **럭키★엔진** (Lucky★Engine) |
| 줄임말(커뮤니티) | 럭시 / 럭키 |
| 저장소·URL | `lucky-sim` 계열 (§5) |

**표기 규칙**: 정식 표기는 `럭키★시뮬레이터`. ★는 장식이 아니라 **인증 표식**이므로 임의로 다른 기호(⭐️, *, ✦)로 바꾸지 않는다. 순수 텍스트 환경(커밋 메시지·로그·코드 주석)에서는 `럭키 시뮬레이터`/`럭키 엔진`으로 쓴다.

## 2. ★ 의 의미 — 브랜드 마크 = 제품 기능

★는 **"엔진이 확정한 사실"의 도장**이다. 우리 UI의 사실 영수증(`FactReceipt`) 헤더에 찍히는 확정 스탬프와 같은 기호이며, 커뮤니티 배지에서도 같은 뜻으로 쓰인다.

- 로고의 ★ = 영수증의 확정 도장 = 배지의 ★ — **셋이 같은 기호, 같은 의미**여야 한다.
- 즉 ★가 붙은 숫자는 **LLM이 지어낸 값이 아니라 엔진이 보증한 값**이다. 이 약속을 UI·문서·마케팅에서 흔들지 않는다.
- 이름("행운")과 엔진(결정론)의 역설은 **의도된 것**이며 슬로건으로 활용한다.

## 3. 문구

- **메인 슬로건**: **"LLM은 이야기를, 엔진은 사실을."**
- 서브: "300턴 뒤에도 골드는 그대로." / "이름만 럭키, 결과는 결정론."
- 정체성 원문(오너): *"리스에서 주구장창 굴리기 힘든 시뮬봇을 쾌적하게 오랫동안 즐길 수 있잖아? 럭키!"*
- **커뮤니티 배지**: `[★ 이 카드, 럭키에서 300턴 디싱크 0]` — 디싱크 하버스가 **실제로 측정**하므로 거짓말할 수 없는 배지다. 측정 없이 이 배지를 붙이지 않는다.
- 호환성 여권(🛂 직역/근사/보존)은 카드별 공유 자산으로 계속 민다.

## 4. 금지

- 제품명·저장소·도메인에 **`Risu` 계열 문자열을 넣지 않는다**(RisuGM·StateRisu 등). 남의 브랜드에 얹는 것이며, 커뮤니티에서 정당하게 비판받을 일이다. 우리는 "리스 호환"을 **설명**하지 브랜드로 **차용**하지 않는다.
- 측정하지 않은 성능·무결성 주장을 배지·문구로 쓰지 않는다.

## 5. 리네이밍 실행 계획 (3단 분리)

리네이밍 폭발 반경 실측: 사용자 표면 16파일 / 내부 `@simbot/*` 78파일 / 영속 계약 문자열 5종.

### A. 사용자 표면 — 지금 (베타 전이라 비용 0)
- 앱 타이틀·랜딩 문구·README·docs·favicon(★)·리포지토리 설명.
- **URL**: Firebase에 새 사이트(`lucky-sim.web.app` 등) 추가, 기존 `simbot-simulator.web.app`은 **리다이렉트로 유지**(죽이지 않는다).

### B. 영속 계약 문자열 — 지금, 단 하위호환 유지
`simbot-play-session/0.1` · `simbot-chat-index/0.1` · `simbot-compile-report/0.1` 등은 **저장 파일의 와이어 포맷**이다. 지금 바꾸지 않으면 사용자가 쌓인 뒤엔 영영 못 바꾼다(마이그레이션 지옥). 지금은 저장분이 오너 것뿐이므로 **바꿀 유일한 기회**.
- 새 계약: `lucky-session/1.0` 등.
- **반드시 구 계약도 읽어준다**(restore 시 옛 문자열 허용 → 저장 시 새 문자열로 기록). 무결성 해시 계약(`sessionIntegrity`의 해시 대상)은 건드리지 않는다.
- localStorage `simbot.llm` → `lucky.llm` (없으면 구 키에서 1회 이관).

### C. 내부 패키지 네임스페이스 — 나중에 일괄
`@simbot/*` → `@lucky/*` (78파일). 사용자 이득 0의 기계적 변경이므로, **코덱스 작업이 없는 유휴 타이밍에 한 번에** 한다. 진행 중 작업과 겹치면 머지 지옥이 된다.

## 6. 시각 (초안)

- 바탕: 현행 리스풍 다크 유지(이사 비용 0 원칙 — 리스 유저가 낯설지 않아야 한다).
- 액센트: ★ 는 앰버/골드. 영수증의 확정 스탬프와 같은 색.
- 모티프: **영수증** — 서사가 위에서 흐르고, 아래에서 확정 사실이 영수증으로 찍힌다. (펜타 수다에서 세 단원이 독립적으로 같은 은유에 도달했다.)
- 서사 본문은 세리프 유지(가독성·몰입 — 오너 확정).
