# 보류된 플랫폼 로드맵 (2026-07-18)

상태: **보류**

제품 오너 결정에 따라 현재 최우선 작업은 `소녀전선: 잔불` 플레이 경험 완성이다. 이 문서는 2026-07-18 저장소 전수 점검에서 도출됐으나 당장 착수하지 않기로 한 플랫폼 차원의 후속 과제를 잃지 않기 위해 기록한다.

## 보류 사유

- 소녀전선 모바일·채팅·에셋·BGM·일일 루프·전투·관계 설정의 실제 사용 피드백을 먼저 해결한다.
- 아래 과제는 소녀전선 완성 작업을 가로막지 않으며, 오너가 재개를 승인하기 전에는 구현 우선순위로 올리지 않는다.

## 보류 항목

1. ROADMAP·BACKLOG·DESIGN의 완료 상태와 최신 코드 정합성 복구
2. Firebase 실배포 전 Playwright E2E 필수 게이트 도입
3. 소녀전선 전용 화면을 부대 편성·시설·지도·생산 대기열·상점·관계 판정 등 재사용 위젯으로 점진 분해
4. DOMINIUM을 새 전용 Svelte 화면 없이 연결하는 범용성 시험
5. Tesselia류 일반 카드의 장기 저장·롤백 검증과 안전 선언형 트리거 opt-in 배선
6. 카드·전역·채팅·모듈·페르소나 로어 우선순위 완성
7. 첫 화면, 호환성 여권, 세션 건강 패널 등 제품 UX 정리
8. 메인 번들 코드 분할과 Tauri 실제 설치·SQLite 검증

## 재개 조건

- 소녀전선 피드백 묶음이 구현·검증되어 오너가 플레이 완성도를 승인했을 때
- 또는 소녀전선 작업 중 구조적 문제 때문에 위 과제 중 하나를 선행해야 한다고 오너가 판단했을 때
